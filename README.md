## Machine Learning & AI Projects Portfolio

# Overview
This repository contains projects focused on **Machine Learning and AI application development**, covering both model building and practical deployment.
Each task demonstrates a different aspect of working with AI systems, from building predictive models on structured data to developing interactive applications using modern AI models.

The repository is organized into separate tasks, with each task containing its own implementation and documentation.


## Projects

## Task 1 - Classification Model on Structured Data

This task focuses on solving a supervised machine learning classification problem.

The objective was to predict the target variable `Transported` using a structured dataset containing a mix of categorical and numerical features.

The project involves:
- Data understanding and preprocessing  
- Handling missing values  
- Feature transformation  
- Model development using CatBoost  
- Proper validation using group-based strategies  

The final solution demonstrates a complete machine learning pipeline, from raw data processing to model evaluation.

## Task 2 - AI Chatbot with Image Support 

This task focuses on building an interactive AI application using local models.

The application allows users to:
- Engage in text-based conversations  
- Upload images and receive AI-generated descriptions  

The project integrates multiple AI capabilities into a single interface using Streamlit and Ollama.

Technologies used:
- Python  
- Streamlit  
- Ollama  
- Llama 3  
- LLaVA  
- Pillow  

The result is a simple multimodal AI system that combines conversational AI with image understanding.
